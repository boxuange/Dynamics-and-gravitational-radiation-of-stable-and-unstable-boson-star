# Data for *Dynamics and gravitational radiation of stable and unstable boson stars*

This repository contains the data associated with the article

**Bo-Xuan Ge et al.**, *Dynamics and gravitational radiation of stable and unstable boson stars* (Physical Review D).

An earlier preprint version of this work appeared on arXiv under the title  
*Hair is complicated: Gravitational waves from stable and unstable boson-star mergers* (arXiv:2410.23839).

The data provided here were used to generate the figures and results presented in the published article.
